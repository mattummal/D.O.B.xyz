# [webhouse](https://19091999.xyz)

> Personal webhouse of Prabhjyoth Mattummal.


## Install

```bash
nvm use 16
npm install -g gh-pages --save-dev
npm i
```

## Development

```bash
npm run start
```

## Production

```bash
npm run deploy
```
